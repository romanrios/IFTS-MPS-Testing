## Tests con NUnit en C#

Proyecto de pruebas unitarias usando NUnit.

### Requisitos

- .NET SDK instalado  
- Extensiones de Visual Studio Code:
  - .NET Core Test Explorer  
  - C#  
  - C# Dev Kit  

### Reconstruir el proyecto
```
dotnet restore
```

### Ejecutar las pruebas
```
dotnet test
```
